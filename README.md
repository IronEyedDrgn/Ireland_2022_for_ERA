# Ireland_2022_for_ERA
population data compatible with the Electoral Redistricting App / ERA

Electoral Redistricting App available at https://era.idea.int/#/
and https://github.com/InternationalIDEA/ERA

Most recent compatible shapefile available at
https://www.cso.ie/en/census/census2011boundaryfiles/
as the Electoral Divisions Zip file

this file matches that zip file with population data, reordered for changes made in 2022
